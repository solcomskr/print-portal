एक ऐसा सेटअप बनाओ की कस्टमर हमारे शॉप में लगे कर को स्कैन करेगा एक वेबसाइट खुलेगी वहां पर अपने रोल नंबर और डेट ऑफ बर्थ डालेगा सबमिट डालेगा प्रिंट का ऑप्शन आएगा इसे दबाते ही हमारे प्रिंटर में सिंगल साइड प्रिंट निकल जाएगा और कस्टमर के मोबाइल से वह साइट क्लोज ओके हम स्क्रीन पर आ जाएगा दोबारा प्रिंट के लिए कस्टमर को दोबारा स्कैनर स्कैन करना पड़ेगा

यहाँ आपके पूरे सिस्टम का कोड है — एडमिन सेटअप (QR जनरेट) + कस्टमर पोर्टल (फ़ॉर्म → प्रिंट → ऑटो-क्लोज़) सब एक ही फ़ाइल में:

HTML
Generate index.html

कैसे काम करता है — पूरा फ्लो

एडमिन सेटअप (शॉप मालिक)

  1. फ़ाइल खोलें → एडमिन पैनल दिखेगा
  2. दुकान का नाम, पता, फ़ोन, और URL भरें (जहाँ यह फ़ाइल होस्ट है)
  3. "QR कोड जनरेट करें" दबाएं → QR बन जाएगा जिसमें पूरी दुकान की info encoded है
  4. QR को डाउनलोड या प्रिंट करके शॉप में लगा दें
  5. प्रिंटर को सिंगल-साइड मोड में सेट करें

कस्टमर फ्लो (QR स्कैन करने पर)

स्टेप
क्या होता है
1. फ़ॉर्मरोल नंबर + जन्म तिथि भरता है
2. प्रिव्यूडिटेल्स दिखती हैं + "प्रिंट करें" बटन (pulse एनिमेशन के साथ)
3. प्रिंटwindow.print() चलता है → सिंगल-साइड स्लिप निकलती है
4. ऑटो-क्लोज़प्रिंट बंद होते ही "पूर्ण हुआ" स्क्रीन → window.close() की कोशिश → मैन्युअल क्लोज़ का मैसेज

सुरक्षा फ़ीचर्स

  • बैक बटन ब्लॉक — history.pushState से कस्टमर वापस फ़ॉर्म पर नहीं जा सकता
  • इनपुट वैलिडेशन — खाली रोल नंबर, भविष्य की तारीख, छोटा रोल नंबर — सब ब्लॉक
  • यूनिक रेफरेंस — हर प्रिंट का अलग रेफरेंस नंबर (जैसे PRN-LK5FJ2-A3B7)
  • दोबारा प्रिंट — बिना QR दोबारा स्कैन किए नहीं हो सकता

होस्टिंग

इस फ़ाइल को GitHub Pages, Netlify, या किसी भी स्टैटिक होस्ट पर अपलोड करें, URL कॉपी करें, और एडमिन में वही URL पेस्ट करें। QR कोड बाकी सब handle कर लेगा।

hosting ke liye stap by stap guid karo
2/2

यहाँ तीन सबसे आसान तरीके बता रहा हूँ — कोई भी एक चुनिए:

तरीका 1: GitHub Pages (सबसे लोकप्रिय, पूरी तरह फ्री)

स्टेप 1 — GitHub अकाउंट बनाओ

  1. github.com खोलो
  2. "Sign up" पर क्लिक करो
  3. ईमेल, पासवर्ड, यूज़रनेम डालो
  4. ईमेल में आए वेरिफिकेशन लिंक पर क्लिक करो

स्टेप 2 — नया रिपॉजिटरी बनाओ

  1. ऊपर दाईं तरफ़ "+" आइकन पर क्लिक करो → "New repository" चुनो
  2. Repository name में लिखो: print-portal
  3. "Public" चुना हुआ रहने दो (ज़रूरी है)
  4. "Add a README file" पर टिक करो
  5. "Create repository" बटन दबाओ

स्टेप 3 — HTML फ़ाइल अपलोड करो

  1. रिपॉजिटरी पेज पर "Add file" बटन पर क्लिक करो → "Upload files" चुनो
  2. अपनी HTML फ़ाइल को ड्रैग-एंड-ड्रॉप करो या चुनो
  3. फ़ाइल का नाम index.html रखो (ज़रूरी है)
  4. नीचे "Commit changes" ग्रीन बटन दबाओ

स्टेप 4 — GitHub Pages ऑन करो

  1. रिपॉजिटरी पेज पर "Settings" टैब पर जाओ (ऊपर बीच में)
  2. बाईं तरफ़ मेनू में "Pages" पर क्लिक करो
  3. "Build and deployment" के नीचे "Source" में "Deploy from a branch" चुनो
  4. "Branch" ड्रॉपडाउन में "main" चुनो
  5. फोल्डर "/ (root)" रहने दो
  6. "Save" बटन दबाओ

स्टेप 5 — लिंक कॉपी करो

  1. 1-2 मिनट रुको, पेज रिफ्रेश करो
  2. ऊपर एक बॉक्स दिखेगा जिसमें लिखा होगा:
    text
    https://yourusername.github.io/print-portal/
  3. इस लिंक को कॉपी करो

स्टेप 6 — अपने पोर्टल में सेट करो

  1. कॉपी किए गए लिंक को ब्राउज़र में खोलो
  2. एडमिन पैनल में जाओ
  3. "पेज का पूरा URL" में वही लिंक पेस्ट करो
  4. दुकान की जानकारी भरो
  5. "QR कोड जनरेट करें" दबाओ
  6. QR डाउनलोड करके प्रिंट करो — बस हो गया!

तरीका 2: Netlify (सबसे तेज़, ड्रैग-एंड-ड्रॉप)

स्टेप 1 — फ़ोल्डर बनाओ

  1. डेस्कटॉप पर एक नया फ़ोल्डर बनाओ → नाम print-portal
  2. अपनी HTML फ़ाइल को इस फोल्डर में रखो
  3. फ़ाइल का नाम index.html रखो (ज़रूरी है)

स्टेप 2 — Netlify पर अपलोड

  1. app.netlify.com/drop खोलो
  2. साइनअप करो (Google से लॉगिन — 5 सेकंड)
  3. अपने print-portal फोल्डर को सीधे ब्राउज़र में ड्रैग करो और छोड़ दो
  4. बस! साइट लाइव हो गई

स्टेप 3 — लिंक कॉपी और सेटअप

  1. ऊपर दिखेगा आपकी साइट का लिंक, जैसे:
    text
    https://funny-name-12345.netlify.app
  2. उस लिंक पर क्लिक करो → एडमिन पैनल खुलेगा
  3. URL फ़ील्ड में वही लिंक पेस्ट करो
  4. दुकान की info भरो → QR जनरेट करो → प्रिंट करो

बोनस — अपना नाम रखना (ऑप्शनल)

  1. Netlify डैशबोर्ड में अपनी साइट पर क्लिक करो
  2. "Site settings" → "Domain management" → "Change site name"
  3. जो नाम चाहो डालो, जैसे: mystore-print
  4. नया लिंक बन जाएगा: https://mystore-print.netlify.app

तरीका 3: Tiiny.host (सबसे आसान, 1 मिनट)

स्टेप 1

  1. tiiny.host खोलो

स्टेप 2

  1. "Upload your site" बॉक्स में अपनी index.html फ़ाइल अपलोड करो
  2. नीचे एक लिंक फ़ील्ड में अपनी पसंद का नाम डालो, जैसे: myshop-print
  3. "Upload" बटन दबाओ

स्टेप 3

  1. लिंक मिलेगा: https://myshop-print.tiiny.site
  2. खोलो → एडमिन में यही URL डालो → QR बनाओ → प्रिंट करो

नोट: Tiiny.host फ्री में कुछ सीमाएं हैं (tiiny.site डोमेन, कुछ ads)। पूरी तरह प्रोफेशनल के लिए GitHub Pages या Netlify बेहतर है।

अक्सर पूछे जाने वाले सवाल

Q: क्या फ़ोन से भी प्रिंट होगा? हाँ, लेकिन फ़ोन से सीधा प्रिंट करने के लिए फ़ोन और प्रिंटर एक ही WiFi पर होने चाहिए, या Google Cloud Print / प्रिंटर का मोबाइल ऐप इंस्टॉल होना चाहिए। सबसे अच्छा यही है कि शॉप में एक टैबलेट रखें जो सीधे प्रिंटर से कनेक्ट हो।

Q: प्रिंटर को सिंगल-साइड कैसे सेट करें?

  • Windows: सेटिंग्स → डिवाइस → प्रिंटर → प्रिंटर प्रॉपर्टीज़ → "Duplex" को "Off" या "One-sided" करें
  • प्रिंटर डिस्प्ले: अगर प्रिंटर में स्क्रीन है तो Settings → Duplex → Off

Q: URL बदलने पर क्या होगा? पुराना QR काम नहीं करेगा। नया QR बनाना होगा। इसलिए पहले ही एक अच्छा URL सेट कर लें।

Q: क्या मैं बाद में दुकान का नाम बदल सकता हूँ? हाँ! एडमिन पैनल में नया नाम डालकर नया QR बनाएं और पुराने की जगह लगा दें।
